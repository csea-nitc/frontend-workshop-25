HTML Tags Overview

This document demonstrates various useful HTML tags.

Semantic Tags

<header>

The <header> tag represents introductory content such as headings, logos, or navigation.

<footer>

The <footer> tag usually contains copyright info, links, or author details.

<article>

The <article> tag is used for content that can stand on its own, like blog posts or news articles.

<section>

The <section> tag groups related content together, usually with a heading.
