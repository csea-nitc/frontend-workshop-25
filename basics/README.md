# React Fundamentals - Stage 1

Welcome to the React Fundamentals project! This stage focuses on the basics of web development using HTML, CSS, and JavaScript. Below is an overview of the files included in this project and what you can expect to learn from each one.

## Project Structure

```
react-fundamentals
└── vanilla
    └── stage1
        ├── 01-html-basics.html
        ├── 02-html-tags.html
        ├── 03-css-basics.html
        ├── 04-css-styling.html
        ├── 05-javascript-basics.html
        ├── 06-javascript-counter.html
        ├── styles.css
        ├── script.js
        └── README.md
```

## File Descriptions

- **01-html-basics.html**: An introduction to HTML, covering the basic structure of an HTML document and common tags such as headings, paragraphs, links, and lists.

- **02-html-tags.html**: A deeper dive into various HTML tags, including semantic tags like header, footer, article, and section, as well as form elements like input fields and buttons.

- **03-css-basics.html**: An introduction to CSS, explaining how to link a CSS file to an HTML document and showcasing basic selectors and properties.

- **04-css-styling.html**: More advanced CSS techniques, including the use of classes, IDs, pseudo-classes, and layout properties such as margin, padding, and background.

- **05-javascript-basics.html**: An introduction to JavaScript, covering how to include JavaScript in an HTML document and basic concepts like variables, data types, and functions.

- **06-javascript-counter.html**: A hands-on example of a simple counter implemented in vanilla JavaScript, featuring buttons to increment and decrement the counter.

- **styles.css**: The CSS styles used throughout the HTML files, enhancing the visual presentation of the content.

- **script.js**: The JavaScript code for the counter functionality, including functions to update the displayed counter value.

## How to View the Files

To view the HTML files, simply open them in a web browser. You can double-click on any `.html` file, and it will open in your default web browser. Explore each file to see how HTML, CSS, and JavaScript work together to create interactive web pages.

Happy coding!
